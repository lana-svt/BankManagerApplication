# Банковское консольное приложение на Spring Framework

## Описание проекта
Консольное банковское приложение, разработанное на Java с использованием Spring Framework, предоставляющее базовые функции для управления пользователями и их банковскими счетами.

##  Технологии

- **Java 21**
- **Gradle**
- **Spring Framework 6.1.5**
- **Jakarta Annotations 2.1.1**

## Доступные операции

### USER_CREATE - Создание нового пользователя
**Входные параметры:** логин пользователя  
**Описание:**
- Создает нового пользователя с автоматически назначаемым ID
- Проверяет уникальность логина в системе
- Автоматически создает первый счет с начальным балансом

### SHOW_ALL_USERS - Просмотр всех пользователей
**Входные параметры:** отсутствуют  
**Описание:**
- Выводит список всех зарегистрированных пользователей
- Отображает информацию о счетах каждого пользователя

### ACCOUNT_CREATE - Создание нового счета
**Входные параметры:** ID пользователя  
**Описание:**
- Создает новый счет для указанного пользователя
- Назначает уникальный ID счету
- Устанавливает начальный баланс согласно настройкам

### ACCOUNT_CLOSE - Закрытие счета
**Входные параметры:** ID счета  
**Описание:**
- Закрывает указанный счет
- Переводит остаток средств на первый счет пользователя
- Не позволяет закрыть единственный счет пользователя

### ACCOUNT_DEPOSIT - Пополнение счета
**Входные параметры:** ID счета, сумма пополнения  
**Описание:**
- Увеличивает баланс счета на указанную сумму
- Проверяет корректность введенной суммы

### ACCOUNT_TRANSFER - Перевод между счетами
**Входные параметры:** ID счета отправителя, ID счета получателя, сумма перевода  
**Описание:**
- Переводит средства между счетами
- При переводе между разными пользователями взимается комиссия
- Перевод между собственными счетами бесплатный

### ACCOUNT_WITHDRAW - Снятие средств
**Входные параметры:** ID счета, сумма снятия  
**Описание:**
- Уменьшает баланс счета на указанную сумму
- Проверяет достаточность средств на счете

### EXIT
**Входные параметры:** отсутствуют  
**Описание:**
- Выход из приложения
